# Bus-Ticket-Booking-System
Java project for bus ticket booking with JDBC integration
Smart Bus Reservation System
A robust Desktop Application for streamlined bus fleet management and passenger ticketing.

🛠️ Technical Stack
Language: Java (J2SE)

GUI Framework: Java Swing & AWT

Database: MySQL 8.0

Architecture: Modular DAO (Data Access Object) Pattern

Connectivity: JDBC (MySQL Connector/J)

🚀 Key Modules
Role-Based Auth: Secure login/signup for Admins and Passengers.

Interactive Seat Selection: A dynamic 40-seat color-coded grid for real-time booking.

Admin Command Center: CRUD operations for bus schedules and revenue monitoring.

Passenger History: Filtered booking records and e-ticket generation.

📊 Database Design
Normalization: Fully compliant with 3rd Normal Form (3NF).

Key Tables: Users, Buses, Schedules, and Bookings.

📂 How to Run
Import the database_backup.sql file into your MySQL Server.

Open the project in NetBeans or IntelliJ.

Ensure the mysql-connector-java JAR is added to the libraries.

Run Login.java to start the application.
